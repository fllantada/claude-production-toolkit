# Claude Production Toolkit

Production-tested patterns and tools for Claude Code. Built from real-world usage, not just official docs.

A Claude Code expert agent trained on community repos instead of official documentation. It learns hook patterns, agent structures, and state management from battle-tested open-source projects — then applies that knowledge to audit and improve your own setup.
